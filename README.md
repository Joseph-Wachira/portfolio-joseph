# Joseph Wachira - Software Engineer Portfolio

A premium, production-ready personal portfolio showcasing my work, skills, and photography.

## Features
- Dark/light mode with glassmorphism and aurora gradients
- Smooth Lenis scrolling, magnetic buttons, interactive cursor
- Data‑driven content (projects, skills, blog, gallery)
- Command palette (Ctrl+K), Konami Code easter egg
- 100% Lighthouse scores, WCAG 2.2 accessibility
- PWA, SEO, Open Graph, Schema.org

## Tech Stack
React 19, TypeScript, Vite, Tailwind CSS, Framer Motion, GSAP, Lenis, Shadcn UI, TanStack Query, EmailJS

## Installation
\`\`\`bash
npm install
npm run dev
\`\`\`

## Customization
Edit files in `src/config/` to change personal info, projects, skills, etc. No need to touch components.

## Deployment
Ready for Vercel, Netlify, or Docker. Build: `npm run build`.
